# KPI dashboard v2

Теперь сайт умеет:
- загружать новый Excel-файл каждый день;
- сохранять данные в локальную базу `kpi_dashboard.db`;
- обновлять показатели по выбранной дате;
- хранить историю и строить динамику по магазину.

## Запуск
```bash
pip install -r requirements_kpi_dashboard_v2.txt
streamlit run app_kpi_history.py
```

## Как использовать
1. Открываешь сайт
2. Загружаешь ежедневный KPI-файл
3. Выбираешь дату отчёта
4. Нажимаешь **Сохранить в историю**
5. Сайт обновляет данные и сохраняет историю

## Важно
История хранится в файле `kpi_dashboard.db` рядом с приложением.
