# KPI Dashboard (Streamlit)

Современный дашборд для сравнения KPI магазинов на основе ежедневных Excel-файлов.

## Что умеет
- загрузка ZIP-архива с KPI-файлами по датам
- автоматический разбор листов:
  - `Продажи вчера`
  - `СВОД`
  - `План.Факт.Прогноз Restore`
- выбор города / кода города
- выбор базового магазина
- сравнение в 4 режимах:
  - один магазин
  - все магазины города
  - похожий оборот
  - ручной выбор магазинов
- KPI-карточки
- продажи за последний день
- динамика продаж и чеков
- круговые, линейные, area, scatter и bar-графики
- таблица сравнения
- сравнение факта МТД с планом месяца

## Запуск локально
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Streamlit Cloud
Главный файл приложения:
- `app.py`

Зависимости:
- `requirements.txt`

## Как использовать
1. Открыть приложение
2. В боковой панели загрузить ZIP-архив с ежедневными файлами
3. Выбрать период, город и магазины для сравнения

## Примечание
Приложение уже умеет работать с архивом, где файлы названы по датам в формате `ДД.ММ.ГГГГ.xlsx`.
Если встроенный CSV не подходит или неполный, просто загрузи новый ZIP-архив через интерфейс.
