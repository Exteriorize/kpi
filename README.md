# KPI Dashboard (Streamlit)

Современный дашборд для сравнения KPI магазинов.

## Что умеет
- загрузка KPI-файла (`.xlsx`, `.xls`, `.csv`)
- выбор города
- выбор базового магазина
- сравнение в 4 режимах:
  - один магазин
  - все магазины города
  - похожий оборот
  - ручной выбор магазинов
- KPI-карточки
- продажи за вчера
- динамика продаж и чеков
- круговые, линейные, area, scatter и bar-графики
- таблица сравнения

## Запуск локально
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Streamlit Cloud
Главный файл приложения:
- `app.py`

Зависимости:
- `requirements.txt`

## Примечание
Сейчас в проекте есть демо-данные на случай, если файл не загружен.
После загрузки реального KPI-файла дашборд будет работать по нему.

Если структура KPI-файла отличается, можно расширить словарь распознавания колонок в `ALIASES` внутри `app.py`.
